## 기술적 결정 사항

| 결정 | 이유 |
|---|---|
| `_build_field_map` 키에서 공백 전부 제거 (`''.join(text.split())`) | HTML 작성자마다 '신청기간' / '신청 기간' 등 공백 유무가 달라 미매칭 발생. 정규화로 흡수 |
| `NoDataError` 예외로 페이지 순회 종료 신호 | `get_current_page_notices`를 단순하게 유지하고, 빈 페이지 처리 책임을 호출자(`get_all_notices`)에 위임 |
| 세션 유효성 10분 캐시 (`_last_verified_at`) | 공고 수십 건 연속 조회 시 매 요청마다 `is_session_valid()` HTTP 호출이 발생하는 것을 방지 |
| `timedelta` 직접 비교 (`.seconds` 미사용) | `.seconds`는 전체 경과 시간이 아닌 초 단위 부분만 반환하므로 1시간 이상 경과 시 오작동 |
